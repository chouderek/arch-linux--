##1. 安裝前準備
    *_**[官網下載ISO 安裝映像檔, 然後使用dd指令將iso檔存進USB隨身碟][1]**_  
      指令：# dd bs=4M if=/path/to/archlinux.iso of=/dev/sdx status=progress && sync  

[1]:https://wiki.archlinux.org/index.php/USB_flash_installation_media
